# How to setup sublime text for Competitive programming

1. Install Sublime Text IDE
2. Install GCC and add your bin path to system environmental variable 
    C:\MinGW\bin
3. Open Sublime text and go to tool
  a. Go to tools
  b. Go to Build System
  c. Go to new Build System
  d. Paste this
  {
    "shell_cmd": "g++ -std=c++17 \"$file\" -o \"$file_base_name.exe\" && \"$file_base_name.exe\" < inputf.in > outputf.out",
    "working_dir": "$file_path",
    "path": "C:/MinGW/bin;$PATH",
    "selector": "source.cpp"
}
e. Save with C++17.sublime-build
f. Create folder on desktop and create 3 files
A.cpp, inputf.in, outputf.out

4. Choose your build system before building once
5. Run your Code with ctrl + shift + B

6. To fasten your running execution code
Go to C:\MinGW\lib\gcc\mingw32\6.3.0\include\c++\mingw32\bits path and open CMD for this path

7. Run this command -> g++ stdc++.h -std=c++17
